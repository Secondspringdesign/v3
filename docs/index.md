## How to Contribute Docs

1. **Pick a template** from the [Templates](/secondspring-v3/templates/) page
2. **Copy it** to `docs/` with an appropriate name (see [naming conventions](/secondspring-v3/contributing/#naming-conventions))
3. **Fill in the content** and update the front matter metadata
4. **Submit a PR** for review

See the full [Contributing Guide](/secondspring-v3/contributing/) for doc lifecycle, review process, and best practices.
